Description
===========

Plugins for the [Collectd-Interface][1].

Usage
=====

Clone this repository and start Collectd-Interface daemon with the `--plugin-path` parameter:

    collectd-interface-daemon --port 5000 --plugin-path /var/collectd/plugins
